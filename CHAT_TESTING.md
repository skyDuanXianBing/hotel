# 独立聊天页面测试指南

## 访问方式

### 1. 直接URL访问
```
http://localhost:8091/chat
```

### 2. 从导航菜单访问
在主界面顶部导航栏中点击「智能客服」菜单项

### 3. 从客服按钮跳转
可以在客服弹窗中添加「进入聊天页面」按钮（可选）

## 页面特色功能

### 🎨 界面设计
- **全屏沉浸式聊天体验**：类似微信/QQ的聊天界面
- **渐变背景**：专业的视觉效果
- **响应式设计**：支持不同屏幕尺寸
- **现代化UI**：圆角、阴影、动画效果

### 💬 聊天功能
- **实时对话**：支持与AI智能客服对话
- **欢迎引导**：初次进入显示功能介绍
- **消息历史**：保留对话记录
- **输入提示**：支持Enter发送、Ctrl+Enter换行
- **服务状态**：显示AI服务在线/离线状态

### 🛠️ 交互特性
- **清空对话**：一键清除所有聊天记录
- **返回主页**：快速导航回主系统
- **正在输入指示器**：显示AI正在思考
- **头像识别**：AI客服和用户头像区分
- **时间戳**：每条消息显示发送时间

### 📱 快捷操作
- **工具栏**：表情、图片、文件上传（UI已准备）
- **键盘快捷键**：
  - `Enter` - 发送消息
  - `Ctrl + Enter` - 换行
- **按钮反馈**：hover效果和点击动画

## 测试场景

### 基础对话测试
- 发送"你好"
- 询问"房间管理怎么操作"
- 问"如何设置房价"
- 测试数学问题如"1+1等于几"

### 功能性测试
- 长文本输入测试
- 快速连续发送消息
- 网络断开重连测试
- 清空对话功能

### 用户体验测试
- 页面加载速度
- 滚动流畅度
- 响应式布局（缩放浏览器窗口）
- 移动端适配（F12开发者工具切换设备）

## 技术特点

- **Vue 3 + TypeScript**：现代化前端技术栈
- **Element Plus**：专业UI组件库
- **响应式设计**：适配各种设备
- **WebSocket Ready**：为实时通讯预留接口
- **模块化架构**：易于扩展和维护
