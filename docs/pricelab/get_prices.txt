POST
/get_prices
Allows syncing of PriceLabs prices and settings



This endpoint allows you to retrieve pricing information for properties based on specified parameters.

Parameters
Name	Description
sync *
object
(body)
Example Value
Model
{
  "sync": {
    "listing_id": "string",
    "rate_plan_id": "string",
    "delta_only": true
  }
}
Parameter content type

application/json
Responses
Response content type

application/json
Code	Description
200	
Request was processed. Success will contain the prices and restrictions last synced from PriceLabs. Failure will contain an error
