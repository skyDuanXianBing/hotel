# Widget Script

For implementing the module, users have to add a blank div and script to their own system, which is mentioned below:

{% hint style="info" %}
**NOTE**

We need to whitelist the domain where the widget will be deployed. Please share your domain so we can whitelist it. The widget API will work only after the domain has been whitelisted.
{% endhint %}

1. **Add the Script:**\
   Include the following script in your HTML to load the required JavaScript file:

   ```html
   <script src="https://static.otaswitch.com/JS/script.js"></script>
   ```
2. **Create a Container Div:**\
   Insert a blank div in your HTML and assign it an id of your choice:

   ```html
   <div id="app"></div>
   ```
3. **Invoke the Function:**\
   Call the function exported from the script, passing in the necessary object. This object is mandatory for loading the module.&#x20;

{% hint style="info" %}
**NOTE**

The configuration object is required to properly load and initialize the module.
{% endhint %}

***

### <mark style="color:yellow;">Sample Example</mark>

```javascript
const tmpScriptObj = {
    elementId: "app",  
    type: "channel-Mapping", // Other possible values are "arr", "booking-details", and "mbs"
    appId: "c3RheWlubi5zdWlzc3UuY29t", 
    propertyId: "y77kDhCABRZxRXEn-KLQYBdl7SVb_1o0TCIZV9EYDso", 
    token: "My42LjI0OS4yNDg=_2zCe4UwqClLJ8IPB25fhRXhaiahF2kja",	 
    channelCode: "", 
    language: "en", 
    themeColor: {
        iconColor: "#1e8ce4"       
    }  
}
```

4. Use below mention function to load module:

```javascript
loadScript(tmpScriptObj);
```
